# Cross Sums Boards Creator

The application takes in 1 commandline parameter: 
the number of boards to create per board size.

The boards will be of sizes 4x4, 5x5, 6x6, 7x7, 8x8.

## Running using Maven

`mvn spring-boot:run -Dspring-boot.run.arguments=10`


## Running using run.sh

`./run.sh 10`
